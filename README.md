# docs.Example

以下の様にコードとドキュメントを管理する。

| dir        | description                                                                                  |
|------------|----------------------------------------------------------------------------------------------|
| documents/ | ドキュメント(構成とか、仕様とか)を管理するディレクトリ                                       |
| docs/      | Github docs で公開する用のディレクトリ。documents/ を `npm run build` した物をデプロイする。 |
| src/       | アプリのコードを管理するディレクトリ                                                         |
